A simple Python-based FLAMES game that predicts relationship between two people based on their names.

## 📌 About the Project

This project implements the classic FLAMES game logic using Python.

The program takes two names as input, removes common characters, and calculates the relationship using the FLAMES algorithm.

---

## 💡 FLAMES Meaning

| Letter | Meaning     |
|--------|------------|
| F      | Friends     |
| L      | Love        |
| A      | Affection   |
| M      | Marriage    |
| E      | Enemies     |
| S      | Siblings    |

---

## ⚙️ How It Works

1. Takes two names as input  
2. Removes common letters  
3. Counts remaining characters  
4. Applies FLAMES elimination logic  
5. Displays final relationship  

---

## 🖥️ Example
Enter first name: Rahul

Enter second name: Priya

Relationship is: Love
